# dotfiles

```bash
git clone <repository-url>
cd dotfiles
./install.sh
./install-apps.sh
./macos-defaults.sh
```

- **`install.sh`** — symlinks dotfiles to home directory
- **`install-apps.sh`** — installs Homebrew apps from `Brewfile`
- **`macos-defaults.sh`** — applies macOS system preferences (run once on fresh machine)
